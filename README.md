# PAPI Random Expansion

## Expansions
This is all the placeholders (V1.0.1)

| Placeholder | RANGE |
| ------ | ------ |
| %random_10% | 0 - 10  |
| %random_32% | 0 - 32  |
| %random_64% | 0 - 64  |
| %random_100% | 0 - 100  |
| %random_128% | 0 - 128  |
| %random_256% | 0 - 256  |
| %random_1000% | 0 - 1000  |
| %random_2304% | 0 - 2304  |



If you want any other range you can make an issue
