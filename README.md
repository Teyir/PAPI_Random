# PAPI Random Expansion

## Expansions
This is all the placeholders (V2.0.0)

| Placeholder | Return |
| ------ |------ |
| %random_max_NUMBER% | Int  |
| %random_range_MIN-MAX% | Int  |

# Example
%random_max_150% -> 80
%random_range_10-100% -> 30




If you want any other range you can make an issue
